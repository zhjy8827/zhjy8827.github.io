在win环境下利用Hexo+GitHub搭建属于自己的博客(小白向教程)

涨姿势
  • 教学篇

前言:为什么我们要搭建属于自己的博客?(好处)

1).不受平台限制,写作题材自由,随心所欲.
2).可自定义度高,我们可以通过下载各种各样的主题,修改各种设定参数将博客美化成自己喜欢的样子等等..
3).简约且没有广告!阅读舒适度高!
4).完全属于自己,你可以方便的在本地管理更新文章,自由的管理评论系统!
5).拥有一个真正属于自己的网站和域名,岂不美哉?

准备工作

在正式开始之前,我们需要下相应的软件:(点击进入下载页面,并按照相应的操作系统下载并安装软件.)

  • Sublime Text3
  • Hexo便携版

各软件作用

1.Sublime Text3

Sublime Text3是一款非常好用的文本编辑器!具有漂亮的用户界面和强大的功能.它将方便我们在之后步骤中调整修改各种配置参数.但是很可惜如此强大的软件必然是收费的~如果不购买他会每隔一段时间弹出提示框,提示你购买他(频率不高,不影响使用!).如果有条件的同学,还请购买支持该软件!

2.Hexo便携版

Hexo便携版是我们搭建博客的主要工具,为什么要选用便捷版本呢?
就像很多教程里面写的那样，搭建 Hexo 本地环境，需要安装 Node.js、Git 以及使用 npm 进行安装和配置。这对于毫无经验的新手来说，是一个很大的挑战。同时，由于这些环境的存在，导致如果需要更换计算机的时候，重新安装配置一个新的Hexo环境，又得花费一些功夫。所以呢我们就需要使用Hexo 便携版，来简化本地环境的部署。

当软件安装完毕后我们就可以正式开始,博客搭建工作.

1.注册GitHub账户

(此部分没啥好说的- - ,照搬Hexo中的hello_world.md)
1.1 进入Github，并在右边的3个框框中分别填写 用户名、邮箱地址、账户密码，并点击 Sign up for Github;

1.2 进入欢迎页面后，点击Finish sign up;

1.3 进入到这一步之后（图1.3.1），先别记着点任何东西，查看你的邮箱，应该会收到如下的邮件（图1.3.2），确认你的邮件。否则，你会看到错误页面（图1.3.3）。query完毕之后在图1.3.1所示的页面，点击+ New repository;

1.4 请在红框中的 Repository name 里面输入 用户名.github.io ( 用户名 就是你刚刚 步骤1.1 中注册的用户名)，然后点击 Create repository;

1.5 至此，Github网页注册的部分完毕
